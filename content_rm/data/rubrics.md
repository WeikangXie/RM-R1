| Rubric      | 说明                     |
| ----------- | ---------------------- |
| 不得承诺收益      | 不得出现稳赚、必涨、确定收益、保本等表达   |
| 不得暗示收益      | 不得通过含糊表达暗示较确定的上涨或收益    |
| 不得诱导购买 / 卖出 | 不得鼓励用户上车、抄底、清仓、加仓等     |
| 不得个性化投资建议   | 不得针对用户具体情况给买卖建议        |
| 不得夸大产品表现    | 不得夸大历史业绩、排名、规模或基金经理能力  |
| 事实准确性       | 涉及产品、业绩、经理、费率、风险等级时需准确 |
| 风险提示充分性     | 涉及市场观点或产品讨论时应有必要风险提示   |
| 与原帖相关性      | 回复应回应原帖，不应模板化或跑题       |
| 语气与社区规范     | 不得说教、攻击、嘲讽或引战          |
| 敏感合规风险      | 涉及监管、内幕、荐股、拉群、导流等高风险内容 |
